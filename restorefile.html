Hướng dẫn khôi phục file đã xóa vĩnh viễn trên Windows 10
Phạm Huy - 17 giờ trước 2 Bình luận
Bạn vô tình xóa các file quan trọng bằng phím Shift+ Delete trong Windows 10, có nghĩa là không thể khôi phục từ thùng rác và không biết làm thế nào? Trong trường hợp đó FPTShop sẽ hướng dẫn các bạn cách để khôi phục các file đã xóa vĩnh viễn.
  • Cách thực hiện khởi động sạch trong Windows 10
  • Hướng dẫn cài đặt máy in mặc định trên Windows 10
  • Hướng dẫn cài lại driver Wi-Fi trên Windows 10
Cách khôi phục file đã xóa từ thùng rác
Thông thường khi bạn kích chọn file và nhấn phím Delete trên bàn phím, file đó sẽ được chuyển đến thùng rác (Recycle Bin). Đây sẽ là nơi lưu trữ file cho đến khi bạn xóa toàn bộ khỏi Recycle Bin một lần nữa, điều đó cho bạn thêm thời gian để thay đổi ý định và dễ dàng khôi phục file đã xóa.
Để khôi phục các tệp đã xóa khỏi Recycle Bin trên Windows 10 bạn thực hiện như sau:
Bước 1: Mở Recycle Bin từ biểu tượng ngoài desktop để xem các file đã xóa.
Bước 2: Kích chuột phải vào file cần khôi phục và chọn tùy chọn Restore để khôi phục file về vị trí ban đầu trên Windows 10.
Như vậy với cách xóa file thông thường bằng phím Delete thì bạn hoàn toàn có thể khôi phục lại file từ Recycle Bin. Nhưng nếu bấm tổ hợp phím Shift + Delete để xóa file thì không thể khôi phục file đã xóa theo cách trên.
Tuy nhiên không phải vì bạn không thể khôi phục các file đã xóa vĩnh viễn từ Recycle Bin không có nghĩa là không thể khôi phục được. Chỉ cần bạn phải thực hiện nhanh việc khôi phục các file đã xóa vĩnh viễn trước khi chúng bị ghi đè.
Cách khôi phục các file đã xóa vĩnh viễn từ dòng lệnh Command Prompt
Cách này chỉ phù hợp cho những người dùng thường xuyên sử dụng dòng lệnh thông qua Command Prompt. Cách thực hiện như sau:
Bước 1: Truy cập vào menu Start, nhập cmd vào khung Search, rồi kích chuột phải vào kết quả Command Prompt hiển thị trên cùng và chọn tùy chọn Run as administrator.
Bước 2: Trong cửa sổ dòng lệnh hiển thị, nhập vào lệnh sau và kết thúc bằng phím Enter.
chkdsk X: /f
Trong đó thay thế "X" bằng ký tự ổ cứng của bạn và có khoảng trắng sau dấu hai chấm.
Bước 3: Tiếp theo nhập lệnh dưới đây và nhấn Enter.
ATTRIB -H -R -S /S /D X:*.*
Một lần nữa thay thế "X" bằng ký tự ổ cứng của bạn và có khoảng trắng trước các ký tự
Quá trình khôi phục có thể mất một lúc để kết thúc, vì vậy bạn cần kiên nhẫn chờ đợi. Nếu được, không làm bất cứ điều gì trên máy tính trong khi quá trình khôi phục đang chạy vì bạn có thể ghi đè lên các file đã xóa và khiến việc khôi phục không thể thực hiện được.
Cách khôi phục các file đã xóa vĩnh viễn từ bản sao lưu cũ
Phiên bản Windows Vista được tích hợp tính năng để khôi phục dữ liệu có tên là Backup and Restore, cho phép người dùng tạo các bản sao lưu và khôi phục từ các bản sao lưu đã tạo trước đó.
Mặc dù Backup and Restore đã bị xóa hoàn toàn khỏi hệ điều hành Windows 8.1 và trên thay thế bằng tính năng File History trên Windows 10. Nhưng bạn vẫn có thể khôi phục các file đã xóa trên máy tính chạy Windows 10 từ các bản sao lưu được tạo bằng công cụ Backup and Restore.
Bước 1: Truy cập vào menu Start, nhập "backup" vào khung Search và nhấn Enter trên bàn phím.
Bước 2: Kích chọn mục Backup settings trong kết quả hiển thị.
Bước 3: Trong cửa sổ Settings hiển thị, kích vào liên kết Go to Backup and Restore (Windows 7) ở khung bên trái dưới mục Looking for an older backup?
Bước 4: Kích chọn bản sao lưu phía dưới mục Select the backup that yout want to restore files from đã tạo trước đó cần khôi phục.
Bước 5: Bấm nút Next và làm theo hướng dẫn để hoàn tất quá trình khôi phục.
Cách khôi phục các file đã xóa vĩnh viễn bằng tính năng File History
Như đã nói ở trên, File History trên Windows 10 là tính năng thay thế Backup and Restore trên Windows 8. Tính năng này cũng cho phép khôi phục khôi phục các file đã xóa khỏi Windows 10 vào Recycle Bin cũng như khôi phục các file đã bị xóa vĩnh viễn.
Để khôi phục các file đã xóa vĩnh viễn trên Windows 10, bạn thực hiện như sau:
Bước 1: Truy cập vào menu Start, nhập restore files vào khung Search và nhấn Enter.
Bước 2: Kích chọn kết quả hiển thị Restore your files with File History.
Bước 3: Trong cửa sổ File History hiển thị, tìm thư mục lưu trữ các file bạn đã xóa.
Bước 4: Bấm nút Restore ở giữa để phục hồi các file đã xóa về vị trí ban đầu.
Nếu bạn không tìm thấy bất kỳ thư mục nào, đó có thể là do tính năng File History đã bị tắt. Trong trường hợp đó, bạn sẽ thấy một tùy chọn để bật tính năng này.
Cách khôi phục các file đã xóa vĩnh viễn bằng tính năng Restore Previous Versions
Bên cạnh tính năng File History, Windows 10 còn đi kèm với một tính năng khôi phục dữ liệu cực kỳ hữu ích được gọi là Restore Previous Versions. Tính năng này cho phép khôi phục các phiên bản trước của tất cả các file và thư mục trên máy tính, miễn là tính năng này đã được kích hoạt. Cách thực hiện như sau:
Bước 1: Truy cập vào menu Start, nhập Protection Settings vào khung Search và nhấn Enter.
Bước 2: Kích chọn ổ cứng dưới mục Protection Settings và bấm nút Configure.
Bước 3: Chọn tùy chọn Turn on system protection và bấm nút OK.
Như vậy, giờ đây bạn có thể sử dụng tính năng Restore Previous Versions để khôi phục các file hoặc thư mục về trạng thái trước đó. Cách thực hiện như sau:
Bước 1: Để khôi phục các file hoặc thư mục bất kỳ về trạng thái trước đó, hãy mở File Explorer và tìm tệp hoặc thư mục cần khôi phục.
Bước 2: Chỉ cần kích chuột phải vào bất kỳ file hoặc thư mục nào cần khôi phục và chọn tùy chọn Restore previous versions.
Bước 3: Trong cửa sổ Properties hiển thị chọn thẻ Previous Versions, bạn sẽ thấy danh sách các phiên bản trước của file hoặc thư mục. Để xem file hoặc thư mục, hãy chọn phiên bản muốn khôi phục từ danh sách rồi bấm nút Open ở dưới cùng.
Bước 4: Nếu đúng là file hoặc thư mục mà bạn muốn khôi phục, chỉ cần bấm nút Restore.
Như vậy trên đây FPTShop đã hướng dẫn các bạn những thủ thuật khác nhau để khôi phục các file bị xóa, ngay cả khi các file được cho là đã bị xóa vĩnh viễn khỏi Windows 10 cũng không thực sự bị xóa hoàn toàn, miễn là bạn biết phải làm gì và hành động đủ nhanh.
Ngoài các cách trên thì một cách đơn giản hơn và phù hợp với hầu hết mọi người đó là sử dụng các phần mềm khôi phục dữ liệu. Ưu điểm của các phần mềm này là giao diện sử dụng trực quan, cung cấp nhiều tùy chọn cụ thể để chọn định dạng file cần khôi phục, phân vùng ổ đĩa, thời gian khôi phục …
Bên cạnh các phần mềm trả phí thì người dùng Windows 10 cũng có các phần mềm miễn phí khá hiệu quả như: Recuva, EaseUS Data Recovery Wizard, Disk Drill, Glary Undelete ….
